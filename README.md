Deskripsi Tugas
Program ini merupakan implementasi algoritma sorting lanjutan menggunakan bahasa C, Program dibuat untuk memenuhi tugas Struktur Data dan Algoritma.
Pada program ini, data akan diurutkan menggunakan metode sorting tertentu sehingga pengguna dapat memahami proses pengurutan data secara lebih efektif.

FITUR PROGRAM
Input data angka
Menampilkan data sebelum sorting
Menampilkan data setelah sorting
Implementasi algoritma sorting
Program berbasis bahasa C

ALGORITMA YANG DIGUNAKAN
Program menggunakan algoritma sorting untuk mengurutkan data dari nilai terkecil hingga terbesar atau sebaliknya sesuai implementasi pada program

TUJUAN PROGRAM
Memahami konsep sorting pada struktur data
Melatih implementasi algoritma menggunakan bahasa C
Meningkatkan kemampuan pemrograman dasar
